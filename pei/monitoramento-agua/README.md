# Projeto Extensionista Integrador (PEI)
## Monitoramento de Consumo de Água na Comunidade

### Descrição
Este projeto permite monitorar o consumo de água em diferentes residências, calcular a média mensal e identificar possíveis excessos, ajudando na conscientização sobre o uso responsável da água.

### Funcionalidades
- Cadastrar novas residências (nome do morador e quantidade de moradores)
- Registrar consumo mensal de água
- Calcular consumo médio anual
- Alertar se o consumo ultrapassar 15 m³/mês
- Exibir histórico de residências com cores de destaque
- Visual moderno, responsivo e interativo

### Tecnologias usadas
- HTML
- CSS
- JavaScript

### Como usar
1. Abra o arquivo `index.html` no navegador.
2. Use o menu para cadastrar casas, registrar consumo e visualizar estatísticas.
3. As residências acima da média terão destaque vermelho; residências dentro da média terão destaque verde.
