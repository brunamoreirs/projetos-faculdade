# Jogo da Adivinhação

## Descrição
Este é um jogo de adivinhação desenvolvido para a disciplina de Lógica de Programação. O usuário deve tentar adivinhar um número aleatório escolhido pelo computador entre 1 e 100.

## Funcionalidades
- Gera número aleatório entre 1 e 100
- Recebe palpites do usuário
- Indica se o palpite é maior ou menor que o número secreto
- Contador de tentativas
- Histórico dos palpites em cards
- Layout moderno e responsivo
- Paleta de cores: branco, cinza e preto

## Como jogar
1. Abra o arquivo `index.html` no navegador.
2. Digite seu palpite e clique em **Verificar**.
3. Veja se o palpite está correto, maior ou menor.
4. Continue tentando até acertar.
5. O histórico mostrará todos os palpites e resultados.
