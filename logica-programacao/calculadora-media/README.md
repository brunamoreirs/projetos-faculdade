# Calculadora de Média

## Descrição
Esta é uma calculadora de média escolar desenvolvida para a disciplina de Lógica de Programação. O objetivo é permitir que alunos calculem a média de suas notas de forma simples ou ponderada e acompanhem o histórico dos resultados.

## Funcionalidades
- Média simples ou ponderada
- Validação de notas (0 a 10)
- Resultado com destaque de situação:
  - Aprovado com Mérito (≥9)
  - Aprovado (≥7)
  - Recuperação (≥5)
  - Reprovado (<5)
- Histórico de alunos calculados em cards
- Layout moderno e responsivo
- Paleta de cores: branco, cinza e preto

## Como usar
1. Abrir o arquivo `index.html` no navegador.
2. Inserir o nome do aluno.
3. Escolher entre **Média Simples** ou **Média Ponderada**:
   - **Simples**: apenas insira as notas.
   - **Ponderada**: insira as notas e os pesos correspondentes.
4. Clicar em **Calcular Média**.
5. O resultado será exibido destacado e adicionado ao histórico.

## Tecnologias usadas
